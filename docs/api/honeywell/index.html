pytket-honeywell

pytket-honeywell is an extension to pytket that allows pytket circuits to be executed on Honeywell’s quantum devices.

pytket-honeywell is available for Python 3.7, 3.8 and 3.9, on Linux, MacOS and Windows. To install, run:

pip install pytket-honeywell

Note

Requires a Honeywell Quantum Solutions account. When initialising the backend for the first time, you will be prompted for your username and password.
