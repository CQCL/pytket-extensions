pytket-pysimplex
==================================

.. image:: CQCLogo.png
   :width: 120px
   :align: right

`Pysimplex <https://github.com/CQCL/simplex>`_ is a Clifford simulator optimized
for large sparse circuits.

``pytket-pysimplex`` is an extension to ``pytket`` that allows ``pytket``
circuits to be simulated using ``pysimplex``.

``pytket-pysimplex`` is available for Python 3.8, 3.9 and 3.10, on Linux, MacOS
and Windows. To install, run:

``pip install pytket-pysimplex``
